# Install

This document is for setting up a development environment to work on the Bluehost Plugin. The plugin itself can be installed from our releases page.

## Prerequisites

To work on this plugin, you will need:
* Node
* Yarn or NPM
* A WordPress development environment, such as VVV or Lando.
* A working knowledge of @wordpress JavaScript packages, React, Sass and PHP.

## Getting Started

1. `git clone https://github.com/bluehost/bluehost-wordpress-plugin.git`
2. `cd bluehost-wordpress-plugin`
3. `npm install`
4. `composer install`
5. `npm run build`
